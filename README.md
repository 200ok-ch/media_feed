200ok media feed
================


Setup
-----

Run once after cloneing the repo to setup pre commit hook

    ./setup.sh


Usage
-----

Edit `media.yml` to add items to the feed.

Subscribe to

    https://raw.githubusercontent.com/twohundredok/media_feed/master/feed.xml
